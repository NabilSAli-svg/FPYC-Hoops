# FPYC Basketball — Marketing Website

Public-facing recreation: the page parents and players land on when looking up registration, schedules, and news.

## What's here

- `index.html` — interactive prototype: navigable home page with anchor sections.
- `Header.jsx` / `Footer.jsx` — global chrome.
- `Hero.jsx` — top-of-page registration hero.
- `Programs.jsx` — House / Select / Skills clinic cards.
- `Announcements.jsx` — warning + news banner stack.
- `Schedule.jsx` — upcoming games strip.
- `News.jsx` — recent posts.
- `FaqContact.jsx` — common questions + contact block.

The page is single-page with anchor jumps; sub-pages are not implemented.
